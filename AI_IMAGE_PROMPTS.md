# Eunice Nails AI Image Generation Prompts

This document contains detailed prompts for all images needed for the Eunice Nails demo site. These images should be generated at high quality and then placed in the `/public/images/` folder.

**IMPORTANT NOTES:**

1. **Total Images Needed: 41 images**
   - **24 Product Images** (6 products × 4 length variants each) — When users select different nail lengths, the hero image changes
   - **4 Collection Images** — Featured collections on home page
   - **3 Journal Images** — Blog preview cards
   - **6 Instagram Images** — Instagram feed grid
   - **4 Optional** — Hero banner variations (future use)

2. **All images must be saved at 1.2x resolution** to keep file sizes smaller while still looking sharp on mobile and desktop. Examples:
  - Product squares: 960×960px
  - Collections: 800×960px
  - Journal: 1280×720px
  - Instagram: 800×800px
  - The CSS will scale them down responsively for mobile/tablet display.

3. **Product Images are Key Feature:** Each of the 6 products has **4 length variants** (short, medium, long, extra-long). The product detail page dynamically changes the displayed image as users select different lengths. All 4 length images for each product should use the same background and lighting for consistency—only the nail length varies.

## TECHNICAL NOTES FOR BEST RESULTS

### 1. **File Format & Optimization**
- Save all images as **JPG** (high-quality, good compression)
- Use quality setting 85-90 for JPGs
- Ensure sRGB color profile for web
- Compress using tools like TinyJPG or similar for faster loading

### 2. **Responsive Display**
- All images will be displayed at varying sizes:
  - **Desktop:** Full size or scaled
  - **Tablet:** Medium size (scaling at 75%)
  - **Mobile:** Small size (scaling at 50-100%)
- Generating at 1.2x resolution keeps files smaller while still looking crisp on modern screens

### 3. **Accessibility**
- Alt text will be added by the development team
- Ensure images are clear and descriptive
- High contrast where appropriate

### 4. **Color Consistency**
Key brand colors to incorporate:
- **Rose Nude:** #C9A28A
- **Milky Rose:** #DAB0A4
- **Mocha/Brown:** #8D6A5A
- **Ivory:** #F0E7DA
- **Charcoal:** #1A1A1A
- **Sage/Green:** #7A8B6F
- **Cream:** #FAF4EE

### 5. **Lighting & Styling**
- Use professional product photography lighting
- Soft shadows for depth without harshness
- Ensure nails appear glossy/salon-quality
- Backgrounds should be neutral and non-distracting
- Consistent lighting across all product images

## BATCH GENERATION TIP

When using an AI image generator that supports batch operations:
- **Product images (24 total):** Generate in groups by product:
  - All 4 length variants of Product 1 together for consistency
  - All 4 length variants of Product 2 together
  - ... and so on through all 6 products
  - This ensures the same background, lighting, and styling across all lengths of each product
- **Collection images (4 total):** Generate together
- **Journal images (3 total):** Generate together  
- **Instagram images (6 total):** Generate together

This batch approach ensures:
- ✅ Visual consistency within each product's length variants
- ✅ Same background/marble in all lengths of a product
- ✅ Consistent lighting across all variants
- ✅ Only the nail length changes between variant images
- ✅ Cohesive brand aesthetic throughout the site
